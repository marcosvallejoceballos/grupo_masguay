Configuración de Router y Switch

Día 3 - 20 de mayo de 2026

Realizado por: Pablo Villalvilla, Marcos Vallejo y Alejandro Pezuela.

Documentación de configuración de Router

Paso 1: Crear las interfaces VLAN desde: Interfaces → VLAN → +
  • Name: vlan11
  • VLAN ID: 11
  • Interface: bridge-local
Paso 2: Asignar direcciones IP a la VLAN.
  • Ruta: IP → Addresses → +
  • Address: 172.16.0.1/20
  • Interface: vlan10-equipo
Paso 3: Configurar el servidor DHCP.
  • Ruta: IP → DHCP Server → DHCP Setup
  • Seleccionar interfaz VLAN
  • Definir DHCP Address Space
  • Definir Gateway
  • Definir rango de direcciones
  • Configurar DNS
  • Configurar Lease Time

Verificación DHCP completada correctamente.

Paso 4: Configurar el Bridge (Puerto Trunk).
  • Ruta: Bridge → VLANs
  • Abrir configuración VLAN del Bridge
  • Configurar VLAN 10
  • Tagged: bridge, ether4

Verificación final correcta.

Documentación de configuración de Switch

  • Añadir VLAN 11
  • Seleccionar los puertos necesarios
  • Pulsar en “save” para guardar la configuración
