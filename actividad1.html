Actividad 1

Preparación del Entorno

¿De qué trata esta actividad?

Esta actividad consiste en preparar el entorno de trabajo instalando diferentes sistemas operativos y aplicaciones necesarias para las prácticas.

Se trabajó con Windows 11, Ubuntu Server y herramientas como Visual Studio Code, Cisco Packet Tracer y OCS Inventory.

¿Qué hicimos?

Primero se creó un pendrive booteable utilizando Rufus para instalar Windows 11 en el ordenador asignado.

Después se preparó otro USB booteable con Ubuntu Server para instalarlo en una Raspberry o dispositivo Wyse.

Finalmente se descargaron e instalaron las aplicaciones necesarias para realizar las prácticas del curso.
