Actividad 2

Diseño y Configuración de Red

¿De qué trata esta actividad?

En esta actividad se trabajó el diseño y la configuración de una red informática utilizando subnetting, VLANs y dispositivos de red en Cisco Packet Tracer.

También se realizó un inventario de hardware utilizando una herramienta de software libre para enviar la información a un servidor.

¿Qué hicimos?

Primero se calcularon y documentaron las subredes necesarias partiendo de la dirección IP 172.16.0.0/16.

Después se diseñó una topología funcional en Cisco Packet Tracer utilizando un router, tres switches, un PC y un servidor para cada equipo.

También se configuraron las VLANs y el servicio DHCP en el router para asignar direcciones IP automáticamente.

Finalmente se utilizó una herramienta de software libre para realizar un inventario de hardware y enviar los datos al servidor.
