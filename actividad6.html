Investigación NAS y FTPS

Investigación sobre sistemas operativos NAS, copias de seguridad y aplicaciones FTPS.

Sistemas Operativos NAS

TrueNAS SCALE

  • ZFS y snapshots
  • Docker y máquinas virtuales
  • SMB, NFS y FTP

OpenMediaVault

  • Ligero y sencillo
  • Compatible con Docker
  • Ideal para Raspberry Pi

Unraid

  • Virtualización
  • Interfaz amigable
  • Soporte Docker

XigmaNAS

  • Basado en FreeBSD
  • Muy ligero
  • Estable

Copias de Seguridad

Estrategia 3-2-1

  • 3 copias de los datos
  • 2 almacenamientos distintos
  • 1 copia externa

Snapshots

Recuperación rápida y protección frente a ransomware.

Rsync

rsync -avh /datos/ usuario@nas:/backup/

Aplicaciones FTPS

FileZilla

Cliente gratuito compatible con FTPS y SFTP.

WinSCP

Cliente Windows con automatización y soporte SFTP.

Cyberduck

Compatible con nube y WebDAV.

VSFTPD

sudo apt install vsftpd -y

Recomendaciones Finales

Se recomienda utilizar SFTP en lugar de FTPS por mayor seguridad. 

sudo apt install openssh-server -y
