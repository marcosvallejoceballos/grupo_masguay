Documentación de prácticas Apache

Instalación, configuración y administración de Apache y servicios FTP en Linux.

Configuración Apache

Actualizar servidor

sudo apt update
sudo apt upgrade

Comprobar Apache

sudo systemctl status apache2

Habilitar Apache

sudo systemctl enable apache2

Configuración FTP

Editar directorio HTML

cd /var/www/html

Instalar VSFTPD

sudo apt install vsftpd

Comprobar servicio FTP

sudo systemctl status vsftpd

Configuración VSFTPD

Backup configuración

sudo cp /etc/vsftpd.conf /etc/vsftpd.conf.bak

Reiniciar servicio

sudo systemctl restart vsftpd

Crear usuario FTP

sudo adduser usuarioftp
