Apache y FTP en Linux

Instalación y configuración de Apache, VSFTPD y servicios FTP.

Actualizar servidor

Actualización completa del sistema Linux. 

sudo apt update
sudo apt upgrade

Comprobar Apache

sudo systemctl status apache2

Habilitar Apache

sudo systemctl enable apache2

Editar HTML

cd /var/www/html

Instalar VSFTPD

sudo apt install vsftpd

Reiniciar servicio FTP

sudo systemctl restart vsftpd

Backup configuración

sudo cp /etc/vsftpd.conf /etc/vsftpd.conf.bak

Crear usuario FTP

sudo adduser usuarioftp
